Simple Drawing Tool

Use cases:

During my last quarters as a senior at University of Washington, I had difficult time to explain class materials over the phone with a friend since she can't visualize what I had in my head. Therefore, I created this app that allow users interact on the same drawing canvas in real time.

How it works:

A user can let other users draw on his/her canvas by clicking on "Share your canvas" button. This action would generate a random code. I did not store any generated codes, so I can't check the uniqueness of them , and therefore, there is a chance that the code is being used by someone else. However, this probability is small since the length of a code is 6 digits.
